## 2. ゲーム内容

ドラゴンクエストモンスターズをモチーフにした 3D モンスター捕獲ゲームです。
プレイヤーは迷宮型フィールドを探索し、フィールド上のモンスターに接触するとバトルシーンに遷移します。
バトルで敵の HP をゼロにすると捕獲判定が発生し、成功すれば仲間にしてパーティに加えることができます。
パーティモンスターを切り替えて戦闘に出すことも可能です。

ステージ 1 のボスを倒すとステージ 2 へ進行し、ステージ 2 のボスを倒すとゲームクリアとなります。

---

## 3. ゲームの流れ

```
タイトル画面
    │
    ▼
ゲーム開始（ステージ1：迷宮フィールド探索）
    │
    ├─ モンスターに接触 → バトルシーン
    │       ├─ 勝利 → 捕獲判定 → パーティに追加
    │       ├─ 勝利（ボス）→ ステージ進行
    │       └─ 敗北 → ゲームオーバー
    │
    ├─ ステージ1ボス（目玉 Lv7）撃破 → ステージ2へ
    │
    └─ ステージ2ボス（ドラゴン Lv15）撃破 → ゲームクリア
```

## 4. 操作方法

### キーボード

| キー | 動作 |
|------|------|
| W / A / S / D | プレイヤー移動（カメラ方向に相対） |
| ← / → | カメラ左右回転 |
| ↑ / ↓ | カメラ上下回転 |
| J | ジャンプ |
| スペースキー | 弾発射（バトル中） |
| Tab | パーティメンバー切り替え |
| M | ステータス画面の表示 / 非表示 |

### ゲームパッド（XInput）

| 入力 | 動作 |
|------|------|
| 左スティック | プレイヤー移動（カメラ方向に相対） |
| 右スティック | カメラ回転 |
| A ボタン | ジャンプ / 決定 |
| RT（右トリガー）| 弾発射（バトル中） |
| RB | パーティメンバー切り替え |
| Start | ステータス画面の表示 / 非表示 |

## 5. 実装機能一覧

### フィールド探索

- **迷宮型マップ**: 壁・天井で囲まれたジグザグ通路の迷宮フィールド
- **2ステージ制**: ステージ 1 とステージ 2 で内壁配置が異なる別マップ
- **壁テクスチャ**: タイリング描画による石壁の表現（`kabe.png`）
- **床テクスチャ**: 専用テクスチャ（`yuka.png`）で統一
- **球面座標カメラ**: プレイヤー中心のオービットカメラ（Yaw/Pitch 操作）
- **カメラ相対移動**: カメラの向きを「前」としてプレイヤーが移動
- **AABB 衝突判定**: プレイヤー・モンスターと壁の衝突処理
- **ミニマップ**: 右下にトップダウン視点のリアルタイムマップ表示
- **距離ベーススポーン**: プレイヤーが近づくとモンスターが出現、離れるとリセット

### モンスター（全 5 種族）

| モンスター | フィールド行動 | 索敵範囲 | 特徴 |
|------------|---------------|---------|------|
| スパイダー | 左右に往復巡回 | 8.0 | 序盤の敵、低ステータス |
| オオカミ | ランダム徘徊 | 10.0 | 中盤の敵、バランス型 |
| 目玉 | 上下に浮遊 | 12.0 | ステージ1ボス |
| ロボット | Z方向に行進 | 10.0 | ステージ2の通常敵 |
| ドラゴン | 静止してプレイヤーを注視 | 15.0 | ステージ2ボス、最強 |

- プレイヤーを感知すると**追跡状態**に遷移
- 一定時間追跡しても接触できない場合、巡回に復帰
- モンスターは壁を貫通せず、壁に沿って移動

### バトルシステム

- **1v1 形式**: プレイヤー側（左）vs モンスター（右）
- **背景画像**: バトル専用背景（`battlehaikei.png`）
- **弾攻撃**: スペースキー / RT で弾を発射し、ヒットでダメージ
- **敵の攻撃**: モンスター種族ごとに異なる攻撃パターン
  - 攻撃間隔・弾速・連射数が種族ごとに設定
- **ダメージ計算**: `攻撃力 - 防御力/2`（最低 1 ダメージ保証）
- **HP バー**: 画面上部に表示、残 HP に応じて **緑 → 黄 → 赤** に色変化
- **モンスター捕獲**: HP をゼロにすると種族ごとの確率で捕獲判定が発生
- **戦闘不能**: パーティモンスターが倒されると自動で次のメンバーに切替
- **全滅**: パーティ全員が倒されるとゲームオーバー

### モンスター攻撃パターン

| モンスター | 攻撃間隔 | 弾速 | 連射数 |
|------------|---------|------|--------|
| スパイダー | 1.2 秒 | 6.0 | 1 発 |
| オオカミ | 2.5 秒 | 8.0 | 2 連射 |
| ドラゴン | 3.5 秒 | 10.0 | 1 発 |
| ロボット | 1.8 秒 | 9.0 | 2 連射 |
| 目玉 | 1.5 秒 | 7.0 | 1 発 |

### パーティシステム

- 捕獲したモンスターをパーティに追加
- **Tab / RB** でパーティメンバーを切り替えて戦闘に出す
- 戦闘で得た**経験値**でレベルアップ（HP / ATK / DEF が成長）
- レベルアップ時 HP 全回復
- 画面左上に**パーティアイコン HUD** を表示（戦闘不能は暗転表示）

### モンスターステータス

| モンスター | HP | ATK | DEF | 獲得EXP | 捕獲率 |
|------------|-----|-----|-----|---------|-------|
| スパイダー | 20 | 5 | 3 | 10 | 30% |
| オオカミ | 50 | 15 | 8 | 35 | 20% |
| 目玉 | 35 | 12 | 5 | 25 | 25% |
| ロボット | 80 | 20 | 15 | 50 | 15% |
| ドラゴン | 120 | 30 | 20 | 100 | 捕獲不可 |

### ステータス画面

- M キー / Start ボタンで表示・非表示を切替
- プレイヤーの名前・レベル・HP・ATK・DEF を表示
- パーティメンバー全員のステータスを一覧表示
- 日本語テキスト対応（GDI ベースのテキストレンダリング）

### シーン管理

- **タイトル画面**: スタート / 操作説明の選択メニュー
- **ゲームオーバー画面**: リトライ / タイトルへ戻る
- **ゲームクリア画面**: タイトルへ戻る
- 各画面に暗い半透明パネル付き UI ボタン（視認性確保）

## 6. 技術的こだわり・アピールポイント

### 球面座標カメラシステム
プレイヤーを中心に Yaw（水平回転）と Pitch（垂直角度）で
カメラ位置を球面座標で計算するオービットカメラを実装。
右スティックと矢印キーの両方で操作でき、Pitch には上下の角度制限を設定。

### カメラ相対移動
プレイヤーの移動方向をカメラの前方・右方ベクトルから算出し、
カメラが向いている方向を「前」として移動できるように実装。
キーボード（WASD）とゲームパッド（左スティック）の両方に対応。

### モンスター AI ステートマシン
各モンスターに**巡回・追跡**の状態遷移 AI を実装。
種族ごとに移動パターン（往復、徘徊、浮遊、行進、静止）が異なり、
プレイヤーとの距離に応じて状態が切り替わる。
壁との衝突判定も行い、壁をすり抜けない自然な動きを実現。

### 距離ベースモンスタースポーン
フィールド上のスポーンポイントに対し、プレイヤーが一定距離に近づくと
モンスターが出現する仕組みを実装。離れるとリセットされ再出現する。
ボスモンスターは倒すと再出現しない。

### モンスター捕獲＆パーティ管理
バトルで HP をゼロにしたモンスターを一定確率で捕獲でき、
パーティに編成して次の戦闘で使用可能。
種族ごとに捕獲率・ステータス・成長率が異なり、戦略性を持たせている。

### 迷宮型 2 ステージ制
壁と天井で囲まれたジグザグ通路の迷宮マップを 2 ステージ分用意。
ステージ間でマップデータをポインタ切替で動的に変更する仕組みを実装。

### AABB 衝突判定システム
プレイヤーとモンスターの両方に AABB ベースの壁衝突判定を実装。
衝突面の法線ベクトルに基づいて押し戻し方向を決定し、壁にめり込まない移動を実現。

### GDI ベース日本語テキストレンダリング
Windows GDI を使用してビットマップに日本語テキストを描画し、
DirectX テクスチャに変換する仕組みを実装。
フォントサイズ・テキスト内容によるキャッシュ機能付き。
